QuIT

QuIT is an Android mobile app designed to help users quit their addiction through feedback, encouragement, community, and analytics.


This project was near and dear to my heart as I tried to quit using nicotine during college. Clearly the app didn't work considering my current weekly bill of Zyn pouches... but it was the first large project I was a part of.


I built this app with a team of three others during my time in school, so not all code is my own.


(One team member - Isaiah Keifer - is now a fireman)


The database used for this app is no longer active, so it is no longer useful. However, it is still a good look into my experience and how I've evolved as a developer.
